# Polimorfismo en Java

En este proyecto trabajé varios ejercicios para entender mejor el uso de herencia y polimorfismo en Java.

## Objetivo

El objetivo principal fue comprender que una variable del tipo padre puede apuntar a un objeto del tipo hijo, y que el método que se ejecuta depende del objeto real.

## Ejercicios realizados

Se trabajaron los siguientes ejercicios:

1. Animal y Perro  
2. Figura y Rectangulo  
3. Empleado y Vendedor  
4. Cuenta y CuentaAhorro  
5. Transporte y Avion  
6. Persona y Estudiante  
7. Vehiculo y Motocicleta  
8. Producto y ProductoDigital  
9. Instrumento y Guitarra  
10. Trabajador y TrabajadorPorHora  

En cada ejercicio se creó:
- Una clase base
- Una clase que hereda
- Un método sobrescrito
- Un main para probar el funcionamiento

## Como aplique la herencia

Utilicé la palabra `extends` para que una clase heredara de otra.  
De esta forma la clase hija puede usar los atributos y métodos de la clase padre.

## Como aplique el polimorfismo

En los main utilicé variables del tipo padre, pero asignándoles objetos del tipo hijo.
